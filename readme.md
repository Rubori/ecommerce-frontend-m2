Ecommerce Frontend - MVP Tienda de Plantas (Bubambilias)

Este es el desarrollo de la primera versión (MVP) de una tienda en línea. El proyecto se enfoca en crear una interfaz funcional, responsiva y dinámica utilizando herramientas fundamentales de desarrollo web frontend.

Propósito del Proyecto
El objetivo principal fue construir una tienda que permita navegar por un catálogo de productos, ver detalles específicos y gestionar un carrito de compras básico, asegurando una buena experiencia de usuario tanto en computadores como en dispositivos móviles.

Funcionalidades Principales
Catálogo Dinámico: Los productos (Bugambilias) se generan automáticamente en la página principal usando JavaScript, lo que facilita el mantenimiento del catálogo.

Navegación de Detalles: Al hacer clic en "Ver más", el sitio redirige a una página específica que muestra la información detallada del producto seleccionado.

Carrito de Compras: Implementación de un contador en la barra de navegación que se actualiza al agregar productos.

Persistencia de Datos: Gracias al uso de localStorage, el número de productos en el carrito no se borra si el usuario refresca la página o navega por el sitio.

Tecnologías Utilizadas
HTML5 Semántico: Para una estructura web clara y bien organizada.

Bootstrap 5: Utilizado para el diseño visual, el sistema de columnas (Grid) y componentes como tarjetas (Cards) y menús (Navbar).

JavaScript / jQuery: Para la lógica de programación, el manejo de eventos (clics) y la actualización del contenido sin recargar la página.

Git & GitHub: Control de versiones para registrar el progreso del desarrollo y publicar el código de forma abierta.

Estructura de Diseño (Responsive)
El sitio fue diseñado siguiendo la filosofía Mobile-First:

Vista Móvil: Los productos se apilan en una sola columna para facilitar la lectura.

Vista Escritorio: Los productos se organizan en una grilla de tres columnas para aprovechar el espacio de la pantalla.
